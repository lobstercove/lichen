# Session Handovers

Use this directory for dated handovers when work leaves behind non-obvious context.

## Naming

- `YYYY-MM-DD-short-topic.md`

## Include

- objective
- what changed
- validation already run
- unresolved risks or blockers
- exact next step

Keep handovers concise.
Move durable truths into `memories/repo/current-state.md` or `memories/repo/gotchas.md` instead of repeating them across many dated files.
